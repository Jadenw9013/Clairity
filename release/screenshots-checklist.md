# Chrome Web Store Screenshots Checklist

Before submitting the extension to the Chrome Web Store dashboard, ensure we capture the following 1280x800 screenshots:

1. [ ] **Simple Mode Action**: Screenshot of a user typing a short, vague prompt in ChatGPT with the Clairity "Fix my question" button injected and active.
2. [ ] **Advanced Mode Metadata**: Screenshot showing the Clairity Popup open in Advanced Mode with preset selectors (Category, Format, Tone) visible.
3. [ ] **Lint Tips & Risk Badge**: Screenshot of the prompt diagnostic summary showing actionable lint tips and an "Accuracy risk" badge with its suggested safety add-on.
4. [ ] **Enhanced Result**: A split-screen or sequential screenshot showing the original vague prompt compared against the new, highly-structured enhanced prompt generated by Clairity.
